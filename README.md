# pm-onboarding
## PIM Onboarding Checklist

Eine interaktive, minimalistische Onboarding-Checkliste für PMs im E-Commerce Umfeld.

**Ziel:** Neue Teammitglieder strukturiert durch die ersten Wochen führen – mit Fokus auf PIM-Team, Systemverständnis und Allgemeinem.

---

## ✨ Features

-  Interaktive Checkboxen mit Fortschrittsanzeige  
-  Persistenz via `localStorage` (Fortschritt bleibt gespeichert)  
-  Fortschrittsbalken + Prozentanzeige  
-  Struktur in Phasen (Zugänge, Architektur, Prozesse, etc.)  
-  Completion-State mit visuellem Feedback  
-  Reset-Funktion  
-  Keine Dependencies – läuft komplett ohne Build-Step  

---
